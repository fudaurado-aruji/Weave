# Weave1 アイデア・バックログ

## UI/UX 定義
- [x] **オブジェクト選択時のツールバー制御**:
    - オブジェクトツールバーは、ボードオブジェクトが選択されている時のみ表示する。
    - 「複製」と「削除」機能は、オブジェクトの直上に表示されるフローティングツールバー（ミニツールバー）に配置する。

## データ・永続化
- [ ] **スナップショット機能**: 状態を保存・復元する機能（SUIREN を参考）。
- [ ] **JSON エクスポート/インポート**: キャンバスデータの外部ファイル保存・読み込み。
- [ ] **JSON インポート機能の拡張**: 現在のユーザーが使用しているボードIDに重ならないよう、IDをずらして復元する機能。

## レイヤー・順序
- [ ] **オブジェクトの順番を変更できる機能**: 前面へ移動、背面へ移動など。
- [ ] **マルチレイヤー管理**: 複数オブジェクトまたはエリア選択時に、選択されたオブジェクトの重なり順をリスト表示し、D&D で直感的に順序を変更できる機能。

## プレゼンテーション・表示制御
- [ ] **一括表示/非表示制御**: 選択中のオブジェクト（単体・複数）の表示状態を一括で切り替える機能。
- [ ] **シーケンシャル表示機能**: 各オブジェクトに表示順を設定し、トリガー（クリックやキー操作等）に応じて順番にオブジェクトを出現させる機能。

## テンプレート・アセット管理
- [ ] **オブジェクトテンプレート機能**:
    - オブジェクト単体、あるいはグループ/エリア化されたものの「塗り・枠線・サイズ（アスペクト比）」を保存し、再利用できる機能。
    - 定型的なプレゼンテーション用レイアウト（例：16:9比率の配置）などのフォームとして活用。
- [ ] **テンプレートライブラリ・ビュー**:
    - 保存したテンプレートをテーブルビューやカードビューで一覧・管理できる機能。
- [ ] **ビューモジュール化**:
    - テーブル/カード等の表示形式を汎用的なモジュールとして切り出し、他の管理画面でも使い回せる設計。
