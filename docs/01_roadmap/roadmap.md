# Weave1 開発ロードマップ

## Phase 1: 基盤構築 (コアエンジン & 空間)
- [x] **無限キャンバスの実装**: React Three Fiber (R3F) を用いた広大な描画領域の確保。
- [x] **座標系の確立**: ワールド座標 (X, Y, Z) とスクリーン座標の相互変換ロジック。
- [x] **カメラシステム**: パン、ズーム、および 2D/3D 視点の切り替え基礎。
- [x] **永続化層の統合**: IndexedDB (Dexie.js) を使った、リロードしても消えない保存の仕組み。
- [x] **UI フレームワーク**: Tailwind CSS をベースにした、Canva ライクな操作パネルの基礎。

## Phase 2: オブジェクト・エコシステム (要素 & 操作)
- [x] **基本オブジェクト抽象化**: 全オブジェクトの共通基盤（移動・選択・回転）の実装。
- [x] **図形 (Shape) 要素**: 塗り、枠線、透過設定が可能な矩形・円形などの実装。
- [ ] **インラインテキスト**: オブジェクト内での直接テキスト入力・編集機能。
- [x] **トランスフォーム制御**: 回転、拡大縮小、複数選択、整列ロジックの実装。
- [ ] **「エリア」(フレーム)**: Miro のフレームのような、グループ化と特定範囲へのジャンプ機能。

## Phase 3: 階層構造の宇宙 (Z軸 & ブランチング)
- [ ] **複層レイヤーボード**: ボードごとの「空域 (Z-range)」の定義と所属の自動切り替え。
- [ ] **3D 俯瞰視点**: カメラを自由に回転させ、多層ボードを空間的に把握するモード。
- [ ] **ブランチング**: Git のように、キャンバスの状態を別レイヤー（Z軸）へ「分岐」させる機能。

## Phase 4: 高度な表現力 (コネクタ & メディア)
- [ ] **コネクタ (ライン)**: オブジェクト間を繋ぐ直線・曲線・矢印。図形への吸着。
- [ ] **メディア・インポート**: 画像、Markdown ファイル、Mermaid ファイルの読み込みと表示。
- [ ] **Mermaid 連携**: Mermaid コードを Weave オブジェクトとして再構成する機能。

## Phase 5: ミドルウェア & インテリジェンス (拡張)
- [ ] **ミドルウェア・コンポーネント**: iframe ライクだが、自由な倍率で PWA/HTML を埋め込む機能。
- [ ] **Gemini API 統合**: AI によるコンテンツ生成や、ボード情報の要約支援。
- [ ] **エクスポート**: 画像出力、および Google ドライブへの保存。

## Phase 6: 洗練 (ポリッシュ & PWA)
- [ ] **PWA 完全対応**: オフライン動作、インストール可能な体験。
- [ ] **Premium Aesthetics**: 微細なアニメーション、ガラスモーフィズム、モダンな演出の追加。
- [ ] **デプロイ自動化**: GitHub Actions から Google Cloud Run への自動デプロイ環境の完成。
