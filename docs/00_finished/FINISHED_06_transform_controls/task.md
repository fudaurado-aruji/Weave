# Mission: Transform Controls

オブジェクトをマウスで掴んで移動させ、ハンドルをドラッグしてサイズを変更する機能を実装し、自由なレイアウト編集を可能にする。

- [x] リサイズハンドルの操作によるスケーリング機能の実装（対角/対辺固定リサイズ） <!-- id: 2 -->
