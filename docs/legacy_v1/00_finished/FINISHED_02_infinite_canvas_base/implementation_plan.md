# Implementation Plan - Infinite Canvas Foundation

## 1. 概要
React Three Fiber (R3F) を使用して、Weave1 のコアとなる無限キャンバスのプロトタイプを構築します。2D ホワイトボードのような操作感（パン・ズーム）を Three.js の 3D 空間上で実現します。

## 2. 構築ステップ
1. **Canvas セットアップ**: `src/App.tsx` を修正し、フルスクリーンの R3F Canvas を配置します。
2. **Infinite Grid**: `drei` の `<Grid />` またはカスタムシェーダーを使用して、無限に続くグリッドを描画します。
3. **カメラ制御**: `OrbitControls` を 2D 平面（XY平面）の移動に最適化するように設定、または `MapControls` を使用します。
4. **座標ヘルパー**: マウスカーソルの位置をワールド座標 (x, y, z) として取得し、デバッグ表示するコンポーネントを作成します。

## 3. 期待される効果
- 広大な空間を自由自在に移動できる基盤が完成する。
- 今後のオブジェクト配置のための基準（座標系とグリッド）が確立される。
