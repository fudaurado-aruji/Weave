# Mission: 09_3d_layer_prototype

## 概要
Weaveの真骨頂である「Z軸階層」のプロトタイプとして、2層目（3D Studio層）を実装する。
1層目の2Dホワイトボードと、2層目の3D空間をシームレスに行き来し、3D空間ならではの体験（地平線、自由視点、空中への配置）の基盤を作る。
