# Mission 08: Studio Experience (Navigation & Floating Toolbar)

## 概要
キャンバスの操作性向上（パン・ズーム）と、オブジェクトごとの詳細なスタイリングを可能にするフローティングツールバーの実装。

## タスクリスト

### 1. ナビゲーション・コントロール
- [x] Spaceキー長押しによるパン（Pan）モードの実装
- [x] Ctrl + マウスホイールによるスムーズズームの実装
- [x] 画面左下にズーム倍率 HUD（スライダー & プリセットメニュー）を実装
- [x] 上部ツールバーに Selection / Pan モード切替スイッチを配置

### 2. フローティング・ツールバー (Floating Toolbar)
- [x] オブジェクト選択時に、その直上にツールバーを表示
- [x] 形状変換（シェイプセレクター）の実装（2列グリッドUI）
- [x] 塗りつぶし色・枠線色のカラーピッカー（ポップオーバー）実装
- [x] 不透明度（Opacity）スライダーの実装
- [x] 枠線の太さ（Border Weight）の調整機能（drei Lineによる実描画反映）
- [x] テキスト配置（左・中央・右）の修正と反映
- [x] オブジェクトの重なり順（最前面・最背面）の変更機能
- [x] ロック/解除、複製、削除の基本アクション実装

### 3. UI/UX ブラッシュアップ
- [x] ツールバー操作時のイベント透過（突き抜け）防止
- [x] ロックされたオブジェクトの選択と解除を可能にする
- [x] Canvas のフラットモード適用による正確な発色（#FFFFFFの白さ）
- [x] 付箋作成時のレイヤー（Z軸）管理によるドット透け防止

### 4. 追加・統合タスク
- [ ] DESIGNサイドバー（詳細設定パネル）とのデータ連動プロトタイプ
- [ ] 複数オブジェクト選択時の整列（プロトタイプ検討）

---
## 完了の定義
- キャンバスをパン・ズームで自在に移動できる。
- オブジェクトのスタイル（色、形、線、配置）をフローティングツールバーから迷わず変更できる。
- 配置したオブジェクトが期待通りの色と重なり順で表示される。
