# 変更履歴

このプロジェクトのすべての重要な変更は、このファイルに記録されます。

## [1.2.0] - 2026-01-28

### 追加
- **トランスフォーム制御 (Transform Controls)**:
    - 8点リサイズハンドル（四隅 + 各辺の中点）による高度な変形。
    - 対角/対辺固定のプロ仕様リサイズロジック。
    - 縮小時に基準点を越えた際の「反転拡大（フリップ）」機能。
    - Shift/Ctrl（Cmd）キーによる軸固定移動・アスペクト比維持リサイズ。
    - 高度なカスケード配置ロジック（複製時やパネル追加時に S/4 ずつずらして重ねる）。
- **パフォーマンス最適化**:
    - ドラッグ/リサイズ中に React の状態更新を介さない「Direct Mutation」の採用。
    - 60fps 以上のヌルヌルとした操作感を実現。
    - 操作完了時のみ IndexedDB へ保存する書き込み頻度の最適化。
- **視覚的ブラッシュアップ**:
    - 操作中にハンドルが歪まない「カウンタースケール」メカニズム。
    - 操作中のハンドル表示制限によるノイズ低減。
    - コーナーハンドルのカーソル向きの正確な修正。


## [1.1.0] - 2026-01-28

### 追加
- **無限キャンバスの基礎実装**:
    - R3F による無限描画領域 of 領域の確保。
    - パン・ズーム対応のカメラシステム。
    - カスタムシェーダーによる高精度ドットグリッド（Canva スタイルの間隔 17px 設定）。
- **開発基盤の強化**:
    - SUIREN プロジェクトより Skills (Visual Auditor, Versioning Manager 等) と Workflows を移植し、Weave 向けに最適化。
    - 開発ロードマップの策定。

## [1.0.0] - 2026-01-28

### 追加
- **プロジェクト初期化**:
    - Vite + React + TypeScript のセットアップ。
    - Tailwind CSS, Three.js, React Three Fiber, Zustand, Immer, Dexie.js, PWA (serwist) の統合。
    - ESLint および Prettier の設定。
    - GitHub リポジトリの初期化と初回プッシュの実施。
    - Mission Control システムによるドキュメント管理体制の構築。
