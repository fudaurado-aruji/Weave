# 変更履歴

このプロジェクトのすべての重要な変更は、このファイルに記録されます。

## [1.1.0] - 2026-01-28

### 追加
- **無限キャンバスの基礎実装**:
    - R3F による無限描画領域 of 領域の確保。
    - パン・ズーム対応のカメラシステム。
    - カスタムシェーダーによる高精度ドットグリッド（Canva スタイルの間隔 17px 設定）。
- **開発基盤の強化**:
    - SUIREN プロジェクトより Skills (Visual Auditor, Versioning Manager 等) と Workflows を移植し、Weave 向けに最適化。
    - 開発ロードマップの策定。

## [1.0.0] - 2026-01-28

### 追加
- **プロジェクト初期化**:
    - Vite + React + TypeScript のセットアップ。
    - Tailwind CSS, Three.js, React Three Fiber, Zustand, Immer, Dexie.js, PWA (serwist) の統合。
    - ESLint および Prettier の設定。
    - GitHub リポジトリの初期化と初回プッシュの実施。
    - Mission Control システムによるドキュメント管理体制の構築。
