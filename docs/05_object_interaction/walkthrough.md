# 変更内容確認 (Walkthrough): Object Interaction & Selection

## 1. 成果物
- [ ] 選択機能 (`useStore` 連携)
- [ ] `SelectionBox` (視覚的フィードバック)
- [ ] ツールバーの表示制御と削除機能

## 2. 変更の詳細
- オブジェクトをクリックすると、SUIREN のようなアクセントカラー（Gold等）の枠線が表示されます。
- 画面上部のツールバーは、オブジェクトを選択している間だけ表示されます。
- ゴミ箱ボタンを押すと、選択中のオブジェクトが即座に消去されます。
